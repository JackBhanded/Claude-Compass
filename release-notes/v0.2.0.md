# Claude Compass v0.2.0 — a new look, in-page answering, learning to stay calibrated

> Paste this into the GitHub release body for the `v0.2.0` tag.

A gorgeous redesign, answering right in the dashboard, and the first step toward a
Compass that keeps pointing at *true* north — your real, current working style.
**Nothing your Claude sessions read changes from the learning work** — it's strictly
read-only evidence you can inspect.

### New look

- **Elevated Claude-brew + dark mode.** The dashboard is now frosted glassmorphism over a soft drifting aurora, with gradient accents, a count-up stat row, a calibration progress bar, a sleek (remembered) dark mode, and strong type hierarchy. The double-click app window is restyled to match — stat row, progress, a bold accent question card, soft card shadows, and mode pills.
- **Answer right in the dashboard — no command.** `compass dashboard` starts a tiny local helper, so you can pick a suggested answer or type your own straight in the page and it saves + re-syncs every Claude surface instantly. The app window answers inline too.

### Added

- **Declination, Phase 1 — opt-in, read-only.** Compass can now learn from your own local Claude history without changing anything yet. A lightweight heuristic detector scans the *new* user turns in a transcript, maps telling phrases — "just the code", "drop the emoji", a repeated "no / stop / I said" on one theme — to the matching profile facet, and records them as **evidence** in a small per-facet ledger.
  - Run it with `compass scan`. Add `--dry-run` to see exactly what it *would* record without writing a thing.
  - Every facet has a **mode**: `auto` / `suggest` / `fixed`. `fixed` facets are never touched — no observations, no changes.
  - This is the safe first slice of the full "stay calibrated over time" design (see `DECLINATION.md`). The actual self-correction — silent flips on safe dials, proposals you approve on the rest — arrives in a later phase.
- **Run at startup.** A new "Run at startup" toggle in the tray menu pins Compass to your per-user Windows startup (no admin, no UAC), so it keeps your sessions attuned from the moment you log in.

### Privacy

The scan reads your local transcripts on your machine; the heuristic stage sends nothing anywhere. Later phases that ask your *own* Claude to adjudicate stay within the same trust boundary as using Claude Code at all — no third party, no new endpoint — and remain opt-in and bound by the `pause` kill-switch.

### Get it

Download **`Claude Compass.exe`** below and double-click — no Python, no terminal. (Windows SmartScreen warns on the unsigned exe → **More info → Run anyway**.)
